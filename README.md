
# Weather Tracker: Daily Conditions & Risk Dashboard

## Project Description
Weather Tracker is a web application that provides users with quick access to current weather conditions and daily weather-related risks such as heat index and wind chill.

The application is especially useful for commuters, students, outdoor workers, and anyone planning activities where weather conditions can pose health or safety risks. By combining real-time weather data with calculated risk indicators, Weather Tracker helps users make informed decisions quickly.

## Target Browsers
- Chrome (Desktop & Android)
- Firefox
- Edge
- Safari (macOS & iOS)
