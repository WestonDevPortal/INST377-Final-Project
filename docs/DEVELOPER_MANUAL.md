
# Developer Manual

## Setup
1. Install Node.js (v18+)
2. Run `npm install`
3. Create `.env` file

## Environment Variables
WEATHERSTACK_API_KEY=your_key
SUPABASE_URL=your_url
SUPABASE_ANON_KEY=your_key

## Run
npm start

## API Endpoints
GET /api/weather?location=city
GET /api/history

## Known Issues
Weatherstack free tier limits forecast data
